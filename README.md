A lightweight x64 code lifter for dynamic cross-process function extraction, relocation, and native external execution.



Designed for advanced reverse engineering and security research, this library enables seamless extraction of complex x64 functions from live target processes. By utilizing CFG (Control Flow Graph) tracing and automatic RIP-relative address fixups, it allows you to rip and execute target algorithms (e.g., decryption routines) locally in your external environment without the need for manual algorithm recreation.
